# Big Collection and Analysis of DayZ Expansion Loadouts

This is my collection and analysis of MANY loadout files for use with the [DayZ Expansion mod](https://steamcommunity.com/workshop/filedetails/?id=2116151222) AI system.

I started with a list of every possible JSON file on github that appeared to have a valid loadout format, which came to about 900 files. I then downloaded all of those files and checked the size and hash to eliminate duplicates per-author, which left around 500 files. I then parsed those to determine the contents of each loadout slot, and test if those items were from the vanilla DayZ [types](https://github.com/BohemiaInteractive/DayZ-Central-Economy/blob/master/dayzOffline.chernarusplus/db/types.xml), the DayZ Expansion [types](https://github.com/ExpansionModTeam/DayZ-Expansion-Missions/blob/master/Template/Chernarus/expansion_ce/expansion_types.xml), or the [Namalsk](https://github.com/SumrakDZN/Namalsk-Server/blob/main/Mission%20Files/regular.namalsk/db/types_dzn.xml) and [DeerIsle](https://github.com/johnmclane666/Deerisle-Stable/blob/master/V5.9/empty.deerisle/db/types.xml) custom maps. 

The details of each loadout is the [Expansion Loadout Analysis](https://github.com/goonwolf666/BigCollectionOfExpansionLoadouts/blob/main/DayZ_Expansion_Loadout_Analysis.xlsx) report, along with links to the original author's repository and files. My copy of each loadout, named per author and original filename, is in the [ExpansionLoadout sub-folder](https://github.com/goonwolf666/BigCollectionOfExpansionLoadouts/tree/main/ExpansionLoadouts).

The table below contains a high-level summary of the findings. Note that many of the "unknown types" that I picked up appear to be typos, [hidden vanilla types](https://dayz.fandom.com/wiki/4rd_CR-550_Mag) that I didn't account for, or just cool stuff from [other mods](https://steamcommunity.com/sharedfiles/filedetails/?id=2663169692) that you need to have installed for the loadout to work properly.

Note that I didn't include [my own collection of custom loadouts](https://github.com/goonwolf666/NamalskSnippets/tree/main/Loadouts), which I know definitely include Expansion Mod and Namalsk DZN item types.
